# 📊 Instagram Analytics Dashboard (Power BI)

📌 *This is a portfolio project showcasing my Power BI and data visualization skills.*

---

## 🚀 Overview

This project is an interactive Power BI dashboard designed to analyze Instagram performance metrics and generate actionable insights. It focuses on clean design, structured layout, and effective storytelling.

---

## 📊 Dashboard Preview

![Dashboard](dashboard.png)

---

## 📌 Key Features

* KPI cards for Total Posts, Reach, Impressions, Followers Gained, and Engagement Rate
* Monthly trend analysis of reach using a line chart
* Performance breakdown (Viral, High, Medium, Low) using a donut chart
* Engagement analysis by:

  * Media Type (Reel, Image, Carousel)
  * Content Category
  * Day of the Week
  * Account Type (Brand vs Creator)
* Interactive slicers for dynamic filtering

---

## 🛠️ Tools & Technologies

* Power BI Desktop
* DAX (Data Analysis Expressions)
* Data Visualization & Dashboard Design

---

## 📁 Dataset

* Instagram Analytics dataset (~30,000 rows, 23 columns)
* Includes metrics like reach, impressions, engagement rate, likes, comments, and more

---

## 🎯 Insights

* Identified trends in reach over time
* Compared engagement across different content types
* Analyzed which categories and days perform better
* Evaluated performance distribution of posts

## 🎨 Design Focus

* Clean and modern UI inspired by professional dashboards
